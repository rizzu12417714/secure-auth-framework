🔐 Secure OS Authentication Framework
Course: Operating Systems — CA2 Project

📌 Project Overview
A secure authentication system built using Python Tkinter that demonstrates multi-layered security including password hashing, 
OTP verification, role-based access control, and audit logging.

⚙️ Features
✅ Password Hashing using Bcrypt  
✅ OTP-Based Two-Factor Authentication  
✅ Role-Based Access Control (Admin/User)  
✅ Account Lockout System  
✅ Audit Logs & Activity Tracking  
✅ Interactive Security Dashboard  

🛠️ Tech Stack
Language: Python  
GUI: Tkinter  
Database: SQLite / MySQL  
Security: Bcrypt, SMTP (Email OTP)  

📁 Project Structure
secure_auth_framework/
├── main.py
├── database.py
├── auth.py
├── otp.py
├── gui/
├── utils/

🚀 How to Run
Step 1: Install dependencies  
pip install -r requirements.txt  

Step 2: Run project  
python main.py
